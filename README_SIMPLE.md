# Bioclaw 小白快速开始

## 一句话介绍

**Bioclaw = 一键安装包**，帮你 5 分钟搭好生物科研 AI 环境。

## 安装（只需一步）

```bash
curl -fsSL https://raw.githubusercontent.com/Rowtion/Bioclaw/main/install.sh | bash
```

然后喝杯咖啡，等 5-10 分钟...

## 使用（3 个命令）

```bash
bioclaw start   # 启动
bioclaw stop    # 停止
bioclaw status  # 看状态
```

## 访问（浏览器打开）

- RStudio: http://localhost:8787
- JupyterLab: http://localhost:8888

密码都是：`bioclaw`

## 搞定！开始科研吧 🚀
