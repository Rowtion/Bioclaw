# Benchling Integration Scripts

Scripts for integrating with the Benchling R&D platform.

## Scripts

1. **manage_sequences.py** - Create and update DNA/protein sequences
2. **inventory_operations.py** - Manage samples and containers
3. **export_data.py** - Export data from Benchling

## Setup

1. Set your Benchling API key as environment variable:
   ```bash
   export BENCHLING_API_KEY="your_api_key"
   export BENCHLING_URL="https://your-tenant.benchling.com"
   ```

## Installation

```bash
pip install -r requirements.txt
```
