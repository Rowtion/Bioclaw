# AlphaFold Database Scripts

This directory contains scripts for accessing and analyzing AlphaFold protein structure predictions.

## Scripts

1. **download_structure.py** - Download protein structures by UniProt ID
2. **analyze_confidence.py** - Analyze pLDDT and PAE confidence metrics
3. **batch_download.py** - Batch download structures for multiple proteins

## Installation

```bash
pip install -r requirements.txt
```
