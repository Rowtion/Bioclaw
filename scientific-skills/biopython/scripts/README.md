# Biopython Scripts

Molecular biology and bioinformatics scripts.

## Scripts

1. **sequence_analysis.py** - Analyze DNA/protein sequences (GC content, translation)
2. **ncbi_query.py** - Query NCBI databases via Entrez
3. **blast_search.py** - Run BLAST searches and parse results

## Installation

```bash
pip install -r requirements.txt
```
