# CellxGene Census Scripts

Scripts for querying the CZ CELLxGENE Census single-cell database.

## Scripts

1. **query_expression.py** - Query gene expression data by cell type/tissue
2. **explore_metadata.py** - Explore available datasets and metadata
3. **cell_type_analysis.py** - Analyze specific cell types across tissues

## Installation

```bash
pip install -r requirements.txt
```
