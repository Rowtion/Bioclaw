# Aeon Time Series Machine Learning Scripts

This directory contains practical Python scripts for time series analysis using the aeon library.

## Scripts

1. **time_series_classification.py** - Classify time series data using ROCKET and MiniRocket classifiers
2. **anomaly_detection.py** - Detect anomalies in time series using STOMP algorithm
3. **forecasting.py** - Forecast future values using ARIMA models

## Installation

```bash
pip install -r requirements.txt
```
