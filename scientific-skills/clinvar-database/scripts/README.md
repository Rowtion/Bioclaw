# ClinVar Database Scripts

Scripts for querying and analyzing clinical variant data.

## Scripts

1. **query_variants.py** - Query ClinVar for variants by gene/condition
2. **annotate_vcf.py** - Annotate VCF files with ClinVar classifications
3. **parse_clinvar_xml.py** - Parse ClinVar XML data files

## Installation

```bash
pip install -r requirements.txt
```
