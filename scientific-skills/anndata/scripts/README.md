# AnnData Scripts

Scripts for working with annotated data matrices in single-cell genomics.

## Scripts

1. **create_anndata.py** - Create AnnData objects from various data sources
2. **io_operations.py** - Read/write h5ad files and format conversions
3. **concatenate_datasets.py** - Merge multiple AnnData objects

## Installation

```bash
pip install -r requirements.txt
```
