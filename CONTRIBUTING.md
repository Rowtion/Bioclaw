## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/AIpoch-claw.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make changes and test
5. Commit with clear messages
6. Push and create a pull request

## Skill Contribution Guidelines

When adding new skills to `scientific-skills/`:

- Follow the AIPOCH skill format
- Include clear `SKILL.md` with usage examples
- Add necessary reference files
- Test with Opencode before submitting

## Code Style

- Shell scripts: Follow POSIX compatibility where possible
- Python: PEP 8 style
- Documentation: Clear, concise, with examples

## Commit Messages

Use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation changes
- `refactor:` Code refactoring
- `test:` Adding tests

## Questions?

Open an issue for discussion.

Thank you for contributing!
